# React Toast Notification App

Develop a toast notification system for a website using **React, TypeScript, and TailwindCSS**.

The app allows multiple buttons (`Success`, `Error`, `Invalid`) to trigger notifications that:

- Appear in the **top-left corner** (stacking for multiple clicks)  
- Show a **center message overlay** briefly  
- Are **fully responsive** and interactive  
- Animate smoothly with TailwindCSS transitions  

---

## Features

### HTML, CSS, JS Version
- Top-left toast notifications
- Center overlay messages
- Smooth slide-in and fade-in animations
- Responsive and stackable notifications
- Auto-dismiss after a set timeout

### React + TypeScript + TailwindCSS Version
- Fully TypeScript safe
- TailwindCSS styling only
- Multiple toasts supported simultaneously
- Smooth animations with Tailwind transitions
- Center overlay shows on each click
- Fully responsive and mobile-friendly
- Continuous notifications without page refresh

---

Click the Success, Error, or Invalid buttons to trigger notifications.

Top-left toasts will appear and stack automatically.
